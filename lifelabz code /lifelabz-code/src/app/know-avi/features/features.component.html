Configurable dispense volume from 0.5ml to 2ml

USB-C Power connector for Main Power as a substitute in case of defect in the Power Adapter

Centralized Monitoring Software for multi-facility setups – Mobile and Enterprise

Designed and manufactured in India

Configurable Notifications: Temp Alert set, RFID enable/disable, Refill, Low Battery, High / Low Utilization

Both Cloud and On premises Server option for both application Software and Data collection

Statistical Analysis – To ensure Sanitiser liquid supply based on demand

Wall-Mount, Table-Top and Floor-Stand options
